# Report Structure

Keep the top-level structure stable:

1. 需求重述
2. 需求确认问题或默认假设
3. 数据初步理解
4. 选定的输出方式及理由
5. 结果生成计划
6. 可下载交付物列表

After that, the analytical body should usually continue with:

- Executive Summary
- Data Quality and Structure
- Relationship or Modeling Entry Points
- Recommended Statistical Methods
- Numeric / Category / Time sections when relevant
- Predictive, Segmentation, and Experiment sections when relevant
- External tool assets when available

Use concise bullet summaries before tables or charts.
