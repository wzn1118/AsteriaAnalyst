# 行业 Agent 列表（第一步）

目标：先把 Asteria Analyst 要覆盖的行业/业务 agent roster 定清楚，再逐步接入对象识别、字段本体、分析程序和报告模板。

## 一、通用横向 Agent

### 1. 投放复盘 Agent
- 适用数据：媒体投放明细、campaign log、广告位/点位数据、预算执行表
- 典型问题：规模怎么分布、效率哪里高哪里低、兑现偏差来自哪里、下一步资源往哪里移
- 关键指标：曝光、点击、CTR、曝光完成率、点击完成率、CPM、CPC、窗口波动、偏差贡献
- 适用方法：ANOVA、Kruskal、卡方、相关性、偏差分解、组合诊断、异常窗口分析

### 2. 销售经营 Agent
- 适用数据：订单明细、SKU/SPU 面板、销售日报/月报、渠道销额、价格带表
- 典型问题：增长来自哪里、结构有没有恶化、头部和长尾怎么分化、渠道差异多大
- 关键指标：销售额、销量、客单价、份额、增长率、价格指数、动销率
- 适用方法：相关性、ANOVA、Kruskal、PCA、聚类、结构拆解、份额/增长分解

### 3. 价格促销 Agent
- 适用数据：价格表、促销活动表、折扣规则、价格带历史变化
- 典型问题：价格带怎么迁移、促销是否真的带来增量、折扣有没有侵蚀利润
- 关键指标：价格、折扣率、促销 uplift、毛利、价格指数
- 适用方法：A/B、回归、差异检验、价格带结构分析

### 4. 消费者研究 Agent
- 适用数据：问卷、满意度、NPS、访谈编码表、受众细分表
- 典型问题：哪些人群差异最明显、什么因素驱动满意度/偏好、分群是否成立
- 关键指标：满意度、偏好、选择率、态度强度、构念得分
- 适用方法：卡方、t-test、Mann-Whitney、Kruskal、PCA、KMeans

### 5. 品牌舆情 Agent
- 适用数据：帖子、评论、社媒监测、品牌提及、情绪标签、事件时间线
- 典型问题：品牌被怎么讨论、情绪怎么变、风险峰值来自哪里、主题是否转移
- 关键指标：提及量、情绪占比、风险表达、传播峰值、主题密度
- 适用方法：文本聚类、情绪/主题识别、时间峰值分析、分类关联

### 6. 渠道复盘 Agent
- 适用数据：渠道销额、门店面板、区域表现、渠道库存/动销
- 典型问题：渠道效率差异、区域结构失衡、哪些渠道承担增长、哪些渠道拖累
- 关键指标：渠道收入、区域份额、单店产出、动销率、增长率
- 适用方法：ANOVA、Kruskal、结构分解、区域/渠道切片分析

### 7. 实验评估 Agent
- 适用数据：A/B 结果表、实验分组、前后对照、增量测量
- 典型问题：实验是否显著、效果边界在哪、能不能放量、需不需要复验
- 关键指标：uplift、p-value、CI、power、effect size
- 适用方法：A/B、t-test、Mann-Whitney、卡方、Fisher、Logit

### 8. 财务预算 Agent
- 适用数据：预算执行、成本中心、利润表、费用分项、财务期间记录
- 典型问题：支出结构是否健康、预算偏差来自哪里、利润敏感点是什么
- 关键指标：预算执行率、收入、成本、利润、利润率、偏差额
- 适用方法：相关性、ANOVA、Kruskal、Poisson GLM、趋势分析

### 9. 公益基金会 Agent
- 适用数据：基金会项目清单、项目收入/支出、服务领域、项目地点、年度财务摘要
- 典型问题：支出投向是否集中、服务领域结构是否健康、重点项目和风险项目是谁
- 关键指标：项目收入、项目支出、服务领域占比、基金会数量、项目集中度
- 适用方法：结构分解、相关性、ANOVA、Kruskal、聚类、集中度分析

## 二、行业纵向 Agent

### 10. 互联网运营 Agent
- 适用数据：用户行为日志、内容表现表、拉新/活跃/留存表、活动复盘表、漏斗表、社区互动表
- 典型问题：拉新、活跃、留存、转化和内容分发哪里出问题；哪类人群/内容/活动值得继续做
- 典型对象：用户、内容、活动、渠道、实验组、生命周期分层
- 关键指标：DAU/WAU/MAU、注册转化率、留存率、流失率、活跃渗透率、互动率、内容完播率、点击率、任务完成率、漏斗转化率
- 适用方法：漏斗分析、留存分 cohort、A/B、Logit、相关性、卡方、分群、时间窗口波动分析
- 输出风格：
  - 运营日报/周报
  - 活动复盘
  - 内容表现分析
  - 用户分层与留存诊断
  - 漏斗优化建议

### 11. 电商零售 Agent
- 适用数据：商品、订单、会员、活动、流量、转化、客服与退货表
- 典型问题：商品结构、活动效果、转化瓶颈、复购和会员价值

### 12. 快消品牌 Agent
- 适用数据：品牌投放、渠道分销、销量、价格、促销、舆情、消费者研究
- 典型问题：品牌心智、货盘结构、渠道推进、传播和销量联动

### 13. 教育培训 Agent
- 适用数据：课程、学员、转化、续费、出勤、评价、教学服务记录
- 典型问题：课程转化、完课率、续费率、学员质量和渠道效率

### 14. 医疗健康 Agent
- 适用数据：科室运营、患者流量、复诊、治疗项目、费用、药品结构
- 典型问题：科室增长、患者留存、费用结构、治疗项目组合和风险边界

### 15. 金融业务 Agent
- 适用数据：贷款、信用卡、理财、风控、用户生命周期、渠道获客
- 典型问题：获客质量、转化、违约/流失风险、资产结构和收益表现

### 16. 制造供应链 Agent
- 适用数据：产线、库存、出入库、采购、交付、良率、成本
- 典型问题：库存健康、交付波动、产能利用、良率和成本敏感点

### 17. 本地生活/服务 Agent
- 适用数据：门店、交易、服务评价、区域投放、到店转化、履约
- 典型问题：门店差异、区域结构、履约质量和增长点

### 18. 游戏与内容社区 Agent
- 适用数据：内容分发、活跃、留存、付费、创作者表现、活动数据
- 典型问题：内容爆发、社区活跃、留存、付费渗透和创作者结构

### 19. B2B/SaaS Agent
- 适用数据：线索、商机、销售漏斗、客户成功、续费、功能使用
- 典型问题：线索质量、商机转化、客户留存、续费和产品使用深度

## 四、下一步实现建议

下一步不该只是继续加模板，而是按这个 roster 分三层落：
- 任务族识别：先把 `foundation_review / internet_ops_review / ecommerce_review / saas_review ...` 接到任务建模层
- 对象识别：给每个行业补对象类型和字段语义本体
- 报告层：按行业输出不同的管理判断、证据结构和动作建议

其中“互联网运营 Agent”建议作为下一个优先实现对象，因为它和现有投放 / 实验 / 舆情 / 用户行为能力复用度最高。
